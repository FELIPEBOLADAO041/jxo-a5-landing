# Landing Page JXO — Selecionadora Modelo A

Landing page responsiva com vídeo controlado pelo scroll, seções de venda e formulário de qualificação.

## Rodar localmente

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

O formulário abre o cliente de e-mail com os dados preenchidos. Antes da publicação final, pode ser conectado ao CRM ou endpoint escolhido pela JXO.
